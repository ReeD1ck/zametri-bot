# Notes Bot

Бот для телеграм, сохраняющий заметки, будь это текст, изображение, видео, документ или аудиосообщение. *Стэк:* Node.JS + MongoDB.

```
$ git clone https://github.com/bifot/NotesBot
$ cd NotesBot
$ npm install
$ mongod --dbpath PATH_DATABASE
$ npm start
```
